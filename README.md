## Description

Tic-Tac-Toe backend with Nest.js & Socket.io
with console Client

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### important

```bash
You can use numbers from 0 to 8 to make moves.
Enjoy... =)
```
